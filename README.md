# Library for mh_z19b
Library for work with mh-z19 CO2 senser.

## Installation
Download and install the libraries to use them in Python.

#### For Python:
`pip install -r requirements.txt`

#### For Python3:
`pip3 install -r requirements.txt`

## How to use
```python
import mh_z19b

print("CO2:", mh_z19b.read())
```
